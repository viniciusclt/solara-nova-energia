Processo Alternativo (Elementos HTML Comuns)

Este exemplo mostra como elementos HTML comuns podem ser convertidos em fluxograma:

Etapa 1: Planejamento

Definir objetivos e escopo do projeto

  • Analisar requisitos
  • Definir cronograma
  • Alocar recursos

Etapa 2: Execução

Implementar as atividades planejadas

  • Desenvolver funcionalidades
  • Realizar testes
  • Documentar processo

Etapa 3: Finalização

Entregar o projeto e avaliar resultados

  • Validar entregáveis
  • Obter aprovação
  • Encerrar projeto

Instruções de Uso

Para importar este fluxograma:

  1. Salve este arquivo como .html
  2. No editor de fluxogramas, clique em "Importar HTML"
  3. Selecione este arquivo
  4. O sistema irá converter automaticamente os elementos em nós de fluxograma

Formatos Suportados:

  • Específico: Elementos com atributos data-flowchart-type, data-x, data-y
  • Automático: Títulos (h1-h6) → Nós de início
  • Automático: Parágrafos (p) → Nós de processo
  • Automático: Itens de lista (li) → Nós de decisão
